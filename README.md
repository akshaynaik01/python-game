# Python-Game

A fun and engaging Python game built with Pygame. This repository contains a complete game implementation with smooth gameplay, intuitive controls, and engaging mechanics.

## Features

- Clean and modular Python code
- Smooth game mechanics with Pygame library
- Interactive gameplay and user-friendly interface
- Easy-to-understand codebase for learning
- Cross-platform compatibility

## Requirements

- Python 3.7 or higher
- Pygame library

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/akshaynaik01/python-game.git
   cd python-game
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the game:
   ```bash
   python main.py
   ```

## How to Play

- Use arrow keys or WASD to move
- Press spacebar to perform actions
- Collect items to increase your score
- Avoid obstacles and enemies

## Game Structure

```
python-game/
├── main.py           # Main game file
├── game.py           # Game logic
├── player.py         # Player class
├── enemy.py          # Enemy class
├── assets/           # Game assets (sprites, sounds)
├── requirements.txt  # Project dependencies
└── README.md         # This file
```

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move |
| Space | Jump/Action |
| ESC | Quit Game |
